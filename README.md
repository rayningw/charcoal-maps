Charcoal Maps
=============

First time
----------

1. npm install -g gulp
2. npm install
3. npm link $path-to-lum-libs/react/slideout

Development
-----------

1. nvm use v4.5.0   # Issue with a gulp plugin against node v6
1. gulp
2. node app.js

Deployment
----------

1. gulp prod-build
2. Copy files from the public folder to a remote server.
